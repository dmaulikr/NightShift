# NightShift  

##项目描述
本项目是一款专门为熬夜工作的人群设计的一款能够劳逸结合、提高生产力、提高工作效率的iOS应用。预计设计的功能有  

* 轻音乐
* 逗比视频
* 放松小游戏
* 照片墙
* 来点刺激！
* 番茄工作法（未实现）
* 聊天机器人（未实现）
* LBS聊天（我们的应用是纯洁的应用，暂时不做）
